###How To Install and Run:

Install:

* curl -sL https://deb.nodesource.com/setup_6.x | sudo -E bash -
* sudo apt-get install -y nodejs
* sudo ln -s /usr/bin/nodejs /usr/bin/node
* sudo apt-get install npm
* npm install

Run:

* sudo npm start
or
* ./run.sh
